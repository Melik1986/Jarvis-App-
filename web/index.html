Axon Flow AI

Voice-to-ERP AI Orchestrator

Manage inventory, invoices, and operations with voice or text. Axon Flow AI connects LLM providers and ERP services in one secure mobile workflow. You can also connect custom/self-hosted LLMs to keep prompts and conversations inside your company perimeter, while using only provider connection configs for routing.

Web client is informational only. Use Axon Flow AI on mobile to run real ERP and AI actions.

How to Start Using Axon Flow AI

  1. Install the app from the App Store and sign in.
  2. Open Profile - Providers and connect your LLM provider.
  3. Add LLM credentials: provider, base URL, model, and API key (OpenAI/Anthropic/Gemini or custom endpoint).
  4. For private deployments, use a custom/self-hosted LLM endpoint so business prompts and chat history can remain within your company infrastructure.
  5. Open ERP Settings and add your ERP connection: provider, URL, database, login, and API key/password.
  6. Run Test Connection for both LLM and ERP, then start a chat.
