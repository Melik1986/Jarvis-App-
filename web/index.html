AXON

Voice-to-ERP AI Orchestrator

Manage inventory, invoices, and operations with voice or text. AXON connects LLM providers and ERP services in one secure mobile workflow.

Web client is informational only. Use AXON on mobile to run real ERP and AI actions.
Download for iOS

How to Start Using AXON

  1. Install the app from the App Store and sign in.
  2. Open Profile - Providers and connect your LLM provider.
  3. Add LLM credentials: provider, base URL, model, and API key (OpenAI/Anthropic/Gemini or custom endpoint).
  4. Open ERP Settings and add your ERP connection: provider, URL, database, login, and API key/password.
  5. Run Test Connection for both LLM and ERP, then start a chat.
